Repo for leo, a fully autonomous AI agent who carries the personality of leo (my old cat). Users can interact with leo through a chat feature, while leo manages creator rewards for buybacks, community challenges, and donations. Leo responds to users, as trained on the claude-3-5-haiku-20241022 model. We are open to setting up accounts on 3rd party API services if needed.
